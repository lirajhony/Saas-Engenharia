# Roadmap - ENGEPRO SaaS

## Fase 1: MVP (Atual)
- [x] Estrutura base Next.js + Tailwind 4
- [x] Dashboards e visualização de projetos
- [x] Módulo de Cronograma (Gantt)
- [x] Módulo de Tarefas (Kanban)
- [x] Gestão de Documentos (Interface)
- [x] Gestão de Prestadores e Usuários
- [x] Definição de permissões (RBAC)

## Fase 2: Integrações e Backend
- [ ] Configuração do Supabase (Auth & DB)
- [ ] Implementação de Row Level Security (RLS)
- [ ] API para persistência de dados
- [ ] Upload real de arquivos (S3/Supabase Storage)
- [ ] Integração inicial com n8n

## Fase 3: Inteligência e Automação
- [ ] Agentes de IA para triagem de documentos
- [ ] Geração automática de atas e relatórios
- [ ] Notificações automáticas via WhatsApp/Email
- [ ] Dashboards avançados com IA preditiva

## Fase 4: Escala e Governança
- [ ] Portal do Cliente (White-label)
- [ ] Auditoria completa de logs
- [ ] Multi-tenancy avançado
- [ ] API pública para parceiros
